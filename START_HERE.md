# Начать поиск

## Компьютер

1. Положи один CV (`PDF`, `DOCX`, `Markdown` или `TXT`) в `workspace/inbox/`.
2. Напиши: **«Начни поиск работы»**.
3. Ответь на один короткий блок важных вопросов, если каких-то данных нет.
4. Выбери вакансию командой `отклик 2` или попроси `ещё`.
5. Проверь готовые CV и письмо. Отправка — только после твоего явного подтверждения.

Поиск идёт пакетами: сначала быстрые источники и дедупликация, затем ИИ подробно читает только лучшие вакансии. Для нескольких откликов формы сканируются один раз, недостающие вопросы собираются в одно сообщение, а заполнение доводится до последних кнопок отправки.

## Телефон / Android

### Remote — когда нужен настоящий локальный workflow

1. На Windows открой этот репозиторий в Codex и включи Remote.
2. Оставь ПК включённым, онлайн и без сна. Для браузерных откликов Windows должна быть разблокирована.
3. На Android открой ChatGPT -> Remote -> эту же задачу.
4. Пиши те же команды: `подробнее 3`, `отклик 2`, `пропустить 4`, `ещё`.

Так телефон управляет desktop-задачей, которая видит локальный CV, tracker, LaTeX и браузер.

### Work — когда ПК выключен

1. В ChatGPT открой `Settings -> Apps -> GitHub`, подключи GitHub и разреши доступ к приватному репозиторию. Ссылка сама по себе не авторизует доступ.
2. Создай один постоянный Work Project.
3. Один раз загрузи туда актуальный CV и локальный `workspace/MOBILE_BRIEF.md`.
4. Напиши: **«Прочитай `.agents/skills/job-hunt/SKILL.md` из репозитория и продолжи мой поиск. Не считай подготовленную форму отправленной»**.

Work без ПК умеет искать, переводить, сравнивать и готовить тексты. Он не видит локальный `workspace`, tracker, Brave-сессию и локальный LaTeX. Для заполнения сайтов, сборки PDF и обновления локального tracker вернись в Remote.
