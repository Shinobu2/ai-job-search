# Начать поиск

## Компьютер

1. Положи один CV (`PDF`, `DOCX`, `Markdown` или `TXT`) в `workspace/inbox/`.
2. Напиши: **«Начни поиск работы»**.
3. Ответь на один короткий блок важных вопросов, если каких-то данных нет.
4. Выбери вакансию командой `отклик 2` или попроси `ещё`.
5. Проверь готовые CV и письмо. Отправка — только после твоего явного подтверждения.

Поиск идёт пакетами: сначала быстрые источники и дедупликация, затем ИИ подробно читает только лучшие вакансии. Для нескольких откликов формы сканируются один раз, недостающие вопросы собираются в одно сообщение, а заполнение доводится до последних кнопок отправки.

## Телефон / Android

1. Открой ChatGPT Work и создай Project.
2. Подключи GitHub или дай ссылку на этот репозиторий.
3. Загрузи CV в Project один раз — личный CV не хранится в Git.
4. Напиши: **«Используй job-hunt skill из репозитория. Начни поиск работы в Германии»**.

Команды: `подробнее 3`, `отклик 2`, `пропустить 4`, `ещё`.

Если на телефоне видна desktop-задача во вкладке Remote, можно просто продолжить её: контекст останется на компьютере.
Без подключённого компьютера телефон умеет искать, сравнивать и готовить тексты; управление браузером и локальная сборка PDF выполняются в desktop-задаче.
