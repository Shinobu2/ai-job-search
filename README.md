# Germany Job Hunt

Личный AI-помощник для поиска работы в Германии: читает CV, ищет свежие вакансии, объясняет лучшие варианты, готовит адаптированные PDF и ведёт простой трекер. Это не SaaS и не универсальная HR-платформа — проект оптимизирован для одного владельца.

## Самый простой запуск

### На компьютере

1. Положи один CV (`.pdf`, `.docx`, `.md` или `.txt`) в `workspace/inbox/`.
2. Открой репозиторий в ChatGPT Codex/Work.
3. Напиши: **«Начни поиск работы»**.

Помощник поймёт CV, один раз задаст недостающие важные вопросы, запустит существующие источники и покажет 5–8 лучших вакансий.

### На Android

Есть два разных режима.

**Remote — основной режим.** Открой эту же desktop-задачу во вкладке Remote. Компьютер должен быть включён, онлайн и не спать; для управления браузером Windows должна оставаться разблокированной. Remote видит локальный CV, tracker, LaTeX и авторизованный браузер.

**Work — автономный облачный режим.** В `Settings -> Apps -> GitHub` подключи GitHub и разреши приватный репозиторий; одна вставленная ссылка доступа не даёт. Создай Project, один раз загрузи актуальный CV и `MOBILE_BRIEF.md`, затем попроси прочитать `.agents/skills/job-hunt/SKILL.md`. Work может искать, сравнивать и готовить текст, но не видит локальный `workspace`, tracker, Brave-сессию или LaTeX на Windows.

Не коммить CV или `MOBILE_BRIEF.md`: они остаются локальными/загружаются только в выбранный Work Project.

## Управление с телефона

После списка достаточно коротких ответов:

- `подробнее 3` — перевести и подробно разобрать вакансию;
- `отклик 2` — подготовить CV и письмо под вакансию №2;
- `пропустить 4` — больше не показывать этот вариант;
- `ещё` — продолжить поиск без повторов.

Нумерация относится к последнему показанному списку. Отправка отклика всегда требует отдельного явного подтверждения. Заполненная форма у последней кнопки считается `подготовлена`, а не `отправлена`.

## Что происходит внутри

- CV извлекается из PDF, DOCX, Markdown или текста.
- Поиск начинается с FreeHire, Bundesagentur Jobsuche и Personio-страниц работодателей.
- Затем AI расширяет охват через прямые карьерные страницы, Make it in Germany, EURES и крупные агрегаторы, проверяя финальную вакансию у работодателя.
- Дубликаты и сохранение обрабатывает существующий код; соответствие кандидату оценивает сама модель без формулы или фальшивого score.
- Для выбранной вакансии используются существующие LaTeX-шаблоны CV и письма.
- Готовая заявка записывается в существующий tracker со статусом `ready_for_review`.

Подробный пользовательский сценарий: [START_HERE.md](START_HERE.md). Инструкции агента: [.agents/skills/job-hunt/SKILL.md](.agents/skills/job-hunt/SKILL.md).

## Локальные команды

```powershell
py tools/extract_cv.py
bun run search:freehire
bun run search:ba
bun run search:employers
```

Требования для полного локального сценария: Python 3.10+, Bun и LaTeX с `lualatex`/`xelatex`. Для PDF-извлечения рекомендуется `pdftotext` (Poppler).

## Приватность и безопасность

`workspace/inbox/` и рабочие результаты не коммитятся. Не клади CV, паспортные данные, разрешение на работу или готовые персональные письма в публичный репозиторий. AI не должен выдумывать опыт или разрешение на работу и не имеет права отправлять отклик без подтверждения.
